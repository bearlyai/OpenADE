# Landing Page — projects/landing

Marketing site for [openade.com](https://openade.com). Single-page, static, deployed to Cloudflare Pages.

## Stack

- **Vite** for bundling (vanilla TypeScript, no framework)
- **Clash Grotesk** (display) + **JetBrains Mono** (code/labels) — loaded from Fontshare/Google Fonts
- All styles in `src/style.css`, all JS in `src/main.ts`
- Static assets in `public/` (served as-is by Vite)

## Commands

```bash
yarn install   # install deps
yarn dev       # dev server on :3000
yarn build     # production build to dist/
yarn preview   # preview production build
```

## Design System

Follows the main app's design language documented in `projects/web/src/_docs/design.md`:

- **True black background** (`#000000`), OLED-friendly
- **No rounded corners** anywhere — all elements are square
- **Orange brand accent**: `oklch(0.702 0.191 41.6)`
- **Borders over shadows** for UI structure (`rgba(255,255,255,0.06)`)
- **1px grid gaps** for card grids (workflow steps, features, powered-by)
- Brand SVG icons sourced from `simple-icons` (GitHub, Apple, Linux) — paths hardcoded in HTML for zero-JS rendering

## Regenerating the OG Image

The OG image (`public/og.png`) is generated by screenshotting `og-template.html` with Playwright.

To regenerate after editing the template:

```bash
# one-time: install playwright and chromium
yarn add -D playwright
npx playwright install chromium

# generate the image
node -e "
const http = require('http');
const fs = require('fs');
const path = require('path');

const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/html' });
  res.end(fs.readFileSync(path.join(process.cwd(), 'og-template.html')));
});

server.listen(9876, async () => {
  const { chromium } = require('playwright');
  const browser = await chromium.launch();
  const page = await browser.newPage();
  await page.setViewportSize({ width: 1200, height: 630 });
  await page.goto('http://localhost:9876', { waitUntil: 'networkidle' });
  await page.screenshot({ path: 'public/og.png', type: 'png' });
  await browser.close();
  server.close();
  console.log('Done: public/og.png');
});
"

# then remove playwright from deps (it's only for image generation)
yarn remove playwright
```

The template must be served over HTTP (not `file://`) because the fonts are loaded from CDN via `@import`.

## Updating Download Links

Download URLs are hardcoded in two places:
1. `index.html` — the download section `<a>` tags
2. `src/main.ts` — the `links` object in `setupDownloadLinks()`

Both must be updated when a new version is released. Search for the current version string (e.g. `v0.52.0`) to find all occurrences.

## OS Detection

The hero CTA and download section automatically detect the visitor's OS and:
- Update the hero button text to "Download for macOS/Linux/Windows"
- Update the hero button `href` to the correct download link
- Highlight the matching download button with an orange border (`.recommended` class)

## SEO

Meta tags are in `index.html` `<head>`. The OG image is referenced as `https://openade.com/og.png`. When updating the OG image, make sure to keep the same filename so social media caches refresh naturally.

Key meta tags: `title`, `description`, `canonical`, `og:*`, `twitter:*`, `theme-color`.
